Database Setup

In order to function, Diogenes needs to know where on your computer the databases of ancient texts are located. These databases are not distributed with Diogenes; they must be acquired separately. You will need at least one database to use Diogenes effectively.

If you need to access this Database Setup page again later on, you can do so via the File menu.
